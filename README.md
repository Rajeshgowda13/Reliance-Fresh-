
# 🥦 Reliance Fresh — ತರಕಾರಿ ಬೆಲೆ ಪಟ್ಟಿ

A daily vegetable price board for **Reliance Fresh** stores and mandis.
Displays live wholesale rates from Karnataka markets in both **Kannada and English**.

## Features
- 📋 27 vegetables pre-loaded with SKU codes
- 🏙️ Live mandi prices from **Bengaluru**, **Kolar** and **Mysuru** APMC markets
- ✏️ Edit vegetable names (Kannada + English), SKU, unit and price
- 📊 Market comparison strip — see price differences across cities
- 🔄 One-click market switcher (BLR / KLR / MYS)
- 🖨️ Print-ready price list table
- 💾 Auto-saves all changes in the browser
- 📅 Date selector for daily price boards

## How to Use
1. Open `index.html` in any browser
2. Click a market tab to load that city's prices
3. Edit any field directly in the table
4. Click **Print** to produce the daily price list

## Data Sources
Prices sourced daily from [market.todaypricerates.com](https://market.todaypricerates.com)
based on APMC mandi arrivals across Karnataka.

## Tech Stack
Pure HTML · CSS · JavaScript — no frameworks, no dependencies.
Works offline after first load.
